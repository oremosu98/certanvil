Gi0/5 MAC + ARP

Right. No LLDP neighbor here, so the MAC 00d8.61f4.0055 on Gi0/5 maps through ARP to a silent host, the backup server.

Correct answer: SRV-BACKUP. Gi0/5 has no LLDP neighbor; its MAC only resolves by crossing the MAC table with ARP: a host, not infrastructure.

Right. ARP resolves MAC 00d8.61f4.0055 to 10.10.10.55.

Correct answer: 10.10.10.55. That is the IP ARP binds to the Gi0/5 MAC, the only way to address a device LLDP can't see.

Gi0/24 LLDP

Right. LLDP shows SW-FLOOR2 on Gi0/24 with the B (bridge) capability, a switch uplink.

Correct answer: SW-FLOOR2. Gi0/24 is an uplink; LLDP marks the neighbor with the B (bridge) capability.

Right. SW-FLOOR2's mgmt address is 10.10.10.3.

Correct answer: 10.10.10.3, the mgmt address LLDP prints for SW-FLOOR2.
