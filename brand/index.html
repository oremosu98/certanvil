§ 04 · the v5.5.12 rule

Any Fraunces hero number MUST force lining-nums. Old-style figures (Fraunces's default in some contexts) make 5/7/9 full-height with descenders while 1/2/0 sit at x-height — uneven heroes. tabular-nums alone only normalises width, not figure-style — they're independent axes. Always set both.

The anti-slop rules.

§ 07 · do / don't

The locked do-not list. Every reskin enforces these — they're what separates the editorial system from generic-SaaS slop.

× Don't
Purple buttons. #5b4fdb / #7c6ff7 is the #1 AI-tell. Eliminated everywhere; the rebrand watched for fallback var(--accent2, #5b4fdb) leaks specifically.
✓ Do
One bronze accent, theme-adaptive. The single focal action per surface is the bronze fill; everything else is hairline-outlined or quiet ghost.
× Don't
Emoji as decorative icons. The only emoji that survive are functional verdict marks (✓ / × / →) where they carry semantic meaning.
✓ Do
Monoline SVG icons for UI; brand-illustrative SVG (the orange flame, sync arrows) for character moments. Trim <style> blocks before inlining — they leak globally.
× Don't
Card-spam grids. 4+ identical soft-shadow cards in a row is the editorial-premium AI-tell. De-card to hairline rows or replace with real data-viz.
✓ Do
Hairline editorial. Cards = surface + 1px border. The bronze focal NBM is the ONE filled card per surface; everything else is quiet.
× Don't
Em-dashes (—) in copy. Throat-clearing ("Here's what..." / "It's worth noting..."). Vague declaratives ("The implications are significant").
✓ Do
Middle-dot (·) for separators. Active voice, specific copy, state directly. Every Analytics card must drive an action (Prescriptive > Descriptive).
× Don't
Hardcoded hex on chrome. Edit styles.css for editorial reskins — the v4.54.x CSS guards assert legacy rules; touching styles.css forces UAT migrations.
✓ Do
Tokens + color-mix(). Add scoped html[data-theme] body overrides in dg-system.css — not UAT-read, no migration, the v5.5.x pattern.

Motion with purpose.

§ 08 · emil-design-eng

Easing: cubic-bezier(0.16, 1, 0.3, 1). Transform + opacity only (GPU). Never scale(0) — entrances start at scale(0.96-0.98). :active{scale(0.97)} press on every clickable. Hover gated to fine pointers. Reduced-motion = fade only.
